UTILS_DEBUG 

Enables debugging output in Utils::_Logger(). This typically includes the calling function's file and line number.

UTILS_RAW 

Raw data output. Skips prepending / appending anything on Utils::_Logger writes (no timestamp, etc).

UTILS_TYPE_ERROR 

Indicates an error and prepends CFG_STR_UTILS_ERROR to Utils::_Logger() output.

UTILS_TYPE_INFO 

Indicates an info message and prepends CFG_STR_UTILS_INFO to Utils::_Logger() output.

UTILS_TYPE_SOCKET
