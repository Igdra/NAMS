Detailed Description

The Server namespace handles all interfacing with the host OS and is the central core of NAMS.

During its boot sequence, the Server will spawn a SocketServer to listen for incoming SocketClient connections, compile and load any Plugin objects, and then indefinitely loop and process game logic until it receives a shutdown notification.
