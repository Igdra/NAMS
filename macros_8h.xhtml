This file contains all macros and certain reference values that "should not" be changed and have hence been omitted from config.h for inclusion here instead.

Definition at line 240 of file macros.h.

#define USLEEP_MAX   1000000

Some platforms limit usleep to 999999. This should not be a problem as long as CFG_GAM_PULSE_RATE is greater than 1. Changing this value will impact the speed at which every action in the game runs. If that is desired then CFG_GAM_PULSE_RATE should be modified instead.
